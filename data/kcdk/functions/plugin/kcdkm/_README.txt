KCDK Manager / KCDKM（標準）
KCDK Manager（データパック）用の拡張機能です。KCDKの有効数を調べて重複していないか確認します。
---------------------------------------------------------------------------------------------------
導入方法：
①「kcdkm.zip」を解凍して〈KCDK 2.0/data/kcdk/functions/plugin/〉に入れてください。
※この拡張機能は既に導入されています。この手順を踏む必要はありません。
※この機能を利用するには『KCDK Manager』（データパック）を導入する必要があります。
