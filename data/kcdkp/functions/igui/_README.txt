Inventory GUI / IGUI（標準）
インベントリにGUIを追加します。『kcdk.admin』チームで使用することができます。
---------------------------------------------------------------------------------------------------
導入方法：
①「igui.zip」を解凍して〈KCDK 2.0/data/kcdk/functions/plugin/〉に入れてください。
②〈KCDK 2.0/data/kcdk/functions/ptp/always.mcfunction〉に「function kcdk:plugin/igui/gate」を追加してください。
③「/function kcdk:plugin/igui/setup」を実行してください。
※この拡張機能は既に導入されています。この手順を踏む必要はありません。

設定方法：
・〈./player_to_run.mcfunction〉で拡張機能を適用するプレイヤーを指定できます。
・「/scoreboard players set igui kcdk.config 0」で拡張機能を無効にします。
・「/scoreboard players set igui kcdk.config 1」で拡張機能を有効にします。
